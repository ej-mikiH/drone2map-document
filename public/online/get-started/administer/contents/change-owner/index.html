所有者の変更

所有者の変更

組織の管理者は、組織のメンバーが所有するアイテムの所有権の変更を行うことができます。

アイテム所有者の変更を行っても ArcGIS Online アイテムの URL、サービス URL は変わりません。そのため、組織サイトに追加している Web マップのリンク修正作業などは必要ありません。

すべてのアイテムの所有権を他のメンバーに一括で変更する場合

  1. 管理者または組織のメンバーが所有するコンテンツの管理権限を持っているユーザーとして ArcGIS Online にサイン インします。

  2. サイト上部の [組織] タブ → [メンバー] タブをクリックします。

  3. 操作を行いたいメンバーの右に表示される […] をクリックして、[コンテンツの転送] をクリックします。

  4. [コンテンツの転送] ウィンドウで、クリックしてアイテムの新しい所有者を選択します。

  5. 必要に応じて既存のフォルダーを選択するか、新しいフォルダーを作成します。

  6. [転送] をクリックします。

[転送] をクリックする前に、所有者が変更されるアイテムのリストを確認するには、[転送するコンテンツ] の下にある [すべてのアイテムの表示] をクリックします。

個別のアイテムの所有権を他のメンバーに変更する場合

  1. 管理者または組織のメンバーが所有するコンテンツの管理権限を持っているユーザーとして ArcGIS Online にサイン インします。

  2. サイト上部の [組織] タブ → [メンバー] タブをクリックします。

  3. 操作を行いたいメンバーの右に表示される […] をクリックして、[アイテムの管理] をクリックします。

  4. メンバーの [マイ コンテンツ] ページへ移動します。

  5. 所有者を変更したいアイテムのチェックボックスをオンにし、[所有者の変更] をクリックします。

  6. [所有者の変更] ウィンドウで、新しい所有者となるメンバーを選択して、[所有者の変更] をクリックします。
